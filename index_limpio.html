⚡ StreamAdmin Pro
Conecta tu base de datos Firebase para sincronizar entre todos tus dispositivos (PC, iPhone, iPad, etc.)
1
Ve a console.firebase.google.com e inicia sesión con tu cuenta de Google. Es completamente gratis.
2
Haz clic en "Crear un proyecto", ponle cualquier nombre (ej: mi-iptv) y sigue los pasos. Desactiva Google Analytics si lo pide.
3
En el panel del proyecto ve a Firestore Database → Crear base de datos. Elige Modo de prueba y la región más cercana (ej: us-east1).
4
Ve a Configuración del proyecto ⚙️ → General → Tus apps. Haz clic en el icono </> (Web), registra la app con cualquier nombre y copia el objeto firebaseConfig.
5
Pega ese código en la pestaña "Configurar Firebase" y haz clic en Conectar. ¡Listo!
💡 Pega el bloque completo firebaseConfig = { ... } que copiaste de Firebase Console

📖 Guía para crear listas M3U desde el Bloc de Notas de Windows
📝 Estructura Básica
#EXTM3U #EXTINF:-1 tvg-name="Canal 1" group-title="Canales",Canal 1 http://servidor.com/stream/canal1 #EXTINF:-1 tvg-name="Película" group-title="VOD",Película http://servidor.com/vod/pelicula.mkv #EXTINF:-1 tvg-name="Serie S01E01" group-title="Series",Serie S01E01 http://servidor.com/series/s01e01.mp4
🏷️ Etiquetas
EtiquetaDescripción
tvg-nameNombre del canal
group-titleCategoría/Grupo
tvg-logoURL del logo
tvg-languageIdioma (Spanish)
tvg-countryPaís (CL)
🖥️ Pasos para crear desde Bloc de Notas
1
Abre el Bloc de Notas de Windows
2
Primera línea: #EXTM3U
3
Por cada canal: una línea #EXTINF + la URL del stream
4
Guarda como milista.m3u (Guardar como → poner extensión .m3u)
5
Importa el archivo en la sección Importar M3U
🐙 Sigue estos pasos para tener tu panel disponible 24/7 gratis desde cualquier dispositivo
📦 Paso 1 — Crear cuenta en GitHub

Ve a github.com → haz clic en Sign up → crea tu cuenta gratis con tu email.

📁 Paso 2 — Crear repositorio

1. Haz clic en el botón verde New o el ícono + arriba a la derecha

2. Nombre del repositorio: iptv-panel (o el que quieras)

3. Selecciona Public

4. Haz clic en Create repository

⬆️ Paso 3 — Subir el archivo

1. En tu repositorio vacío, haz clic en "uploading an existing file"

2. Renombra el archivo de iptv-panel.html a index.html antes de subirlo

3. Arrastra el archivo index.html a la zona de carga

4. Escribe un mensaje como primer commit y haz clic en Commit changes


🌐 Paso 4 — Activar GitHub Pages

1. En tu repositorio ve a Settings (pestaña arriba)

2. En el menú lateral haz clic en Pages

3. En Source selecciona Deploy from a branch

4. En Branch selecciona main y carpeta /(root)

5. Haz clic en Save

6. Espera 1-2 minutos y tu panel estará en:

https://TU-USUARIO.github.io/iptv-panel/

¡Abre esa URL desde tu iPhone y funcionará sin encender el PC! 🎉

💡 Consejo Final
⚠️ Si configuraste Firebase: los datos se sincronizan automáticamente entre todos tus dispositivos. Si usas el modo sin Firebase, los datos son locales de cada navegador.
